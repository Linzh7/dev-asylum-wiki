# 常用词汇表

[[toc]]

## LGBTQIA+

一个囊括与大多数人不同的性取向，性别表达，性别身份，以及间性的缩略词，
其包含，但不限于：女同性恋(**L**esbian)，男同性恋(**G**ay)，双性恋（**B**isexual)，跨性别（**T**ransgender)，酷儿（**Q**ueer），间性人（**I**ntersex），无性恋（**A**sexual） 和/或 无浪漫倾向（**A**romantic），
以及任何与常人不同但未提及的性取向/性别表达/性别身份（**+**）。[^1]

>[!NOTE]
> 任何人的性取向/性别身份是在**形容**那个人在性/性别方面的特征，**而非规定**那个人该怎么样不该怎么样，或是“那个人所认同的性别身份/性取向/性别表达/性征不与定义中的完全一致”这件事本身能够被用来否定那个人的性别身份/性取向/性别表达/性征。

#### 女同性恋(Lesbian) [^1]

可以形容任何认可自己为女性（或任何非男性的性别）而又主要从其ta认可自己为女性（或任何非男性的性别）的人感受到外貌/性/浪漫 和/或 情感吸引的人。

#### 男同性恋(Gay) [^1]

可以形容任何认可自己为男性而又主要从其ta认可自己为男性的人感受到外貌/性/浪漫 和/或 情感吸引的人。

#### 双性恋（Bisexual）[^1]

可以形容任何从不止一个性别的人感受到外貌/性/浪漫 和/或 情感吸引的人。一些对所有的性别/性别身份的人产生外貌/性/浪漫 和/或 情感吸引的人可能将自己形容为“泛性恋”（pansexual）。

有些认同自己为男同性恋或女同性恋的人可能为了人身安全考量而不得不和另一个性别的人进行性行为/结婚，但是仅凭“和异性发生性关系/结婚”这一点并不能使得那个认可自己为同性恋的人成为，或事实上是双性恋。

#### 跨性别（Transgender）[^1]

可以形容任何性别表达/性别身份和出生时被指派的(i.e. 出生证上在出生时所记载的）性别不一致的人的一个词汇，
其包括但不限于：用手术，医学或其它手段，或计划用这些手段使得 自己的身体 有 自己认同的性别所包含的性征 的人，性别身份/性别表达与指派性别不一致但不希望改变自己身体的人，
有多个性别认同或没有性别认同的人，性别身份是流动或会变的人，或是任何性别身份/所做的行为和社会对于该性别所规范的不一致的人。

性别身份和性取向不同，一个认同自己为跨性别的人可能认同自己是异性恋，双性恋，同性恋，无性恋，或酷儿。

一些人可能不用“跨性别”而用“变性人”（Transsexual）来形容自己，而另一些人则可能认为用“变性人”来形容自己是具有冒犯性的。无论ta们选择用什么词语来称呼自己，请尊重ta们的选择。

#### 酷儿（Queer）[^1]

可以形容任何性取向，性别身份，性别表达，性征与社会规范不一致的人。

#### 间性人（Intersex）[^1]

可以形容任何出生时性征与男女二元的典型性征不一致的人（如生殖器，染色体等）（举例：出生时同时有阴茎以及阴道，有阴茎但是第23条染色体为“XX” 等） 。

这些独特的性征可能在出生时就已经显现，或直到青春期或在这之后才显现，也有可能任何人都不知道。

间性这个概念本身不同于性取向，性别身份，性别表达，或跨性别，它更多是形容因为人类的多样性所关联的，人体在生理上出生时的自然差异。

#### 无性恋（Asexual）[^2]

可以形容任何感受不到或感受很少性吸引力 和/或 性欲的人。无性恋是一个光谱，没有任何一个定义能够准确地囊括到所有的无性恋者。

一个无性恋者可能会或不会和其ta人发生性行为 和/或 边缘性行为（e.g. 为了取悦与ta发生性行为的那个人，或从事性工作之类的）。无性恋形容的是一个人感受到的**性吸引力** 和/或 **性欲**的多寡/重要与否，而非一个人实际上有没有能力发生性行为。

一个无性恋者有可能同时是无浪漫倾向者，也有可能和其ta人有亲密关系（浪漫/非浪漫,有性行为/无性行为）；一个无性恋者可能会结婚。

#### 无浪漫倾向（Aromantic） [^2]

跟无性恋类似，无浪漫倾向形容任何感受不到或感受很少浪漫吸引力的人。无浪漫倾向也是一个光谱，也没有任何一个定义能够准确地囊括到所有的无浪漫倾向者。

无浪漫倾向者可能在一段浪漫的亲密关系中但仍然体会不到浪漫吸引，无浪漫者可能会结婚，无浪漫者也有可能会发生性关系。跟无性恋类似，无浪漫倾向形容的是一个人感受到的浪漫吸引的多寡/重要与否，而非一个人实际上有没有能力进入浪漫关系。

浪漫吸引不同于审美吸引，柏拉图式吸引，感官吸引，一个无浪漫倾向者可能也同时是无性恋者。


## 寻求庇护者（Asylum Seeker）

任何正在原籍国之外准备或正在寻求难民保护或准备离开原籍国以寻求难民保护的的人。

一个寻求庇护者可能已经向当前所在的非原籍国的国家当局/联合国难民署提出了难民声请，也有可能因为各种原因想要寻求难民保护但是没办法这么做。


## 新来者（Newcomer）

任何一个在最近才入境一个国家的人，无论ta有没有合法签证以及ta是以什么样的方式进入的 

## 难民声请（Refugee Claim）或 庇护声请（Asylum Claim)（两者为同义词）

任何一个在不是原籍国的国家而又向该国当局/联合国难民署声称自己是《难民公约》中定义的难民并请求该国当局/联合国难民署认可自己为难民的举措

一宗声请(Claim)和申请(Application)不同，举个例子，在一宗移民申请中，若该申请获批，则申请人是在申请获批的那一刻才成为移民的；声请就不同，若一宗难民声请最后获得认可，则大部分情况下是在提出声请的那一刻就已经事实上是难民了，只是在等待被认可成为难民；

但也有一种情况下是在提出难民声请的时候ta并不符合难民的定义，但是，直到ta的声请被考量间中发生了一些事，使得ta在被考量的时候实际上符合难民的定义，那这种情况下ta则是在什么时候开始符合难民的定义了什么时候就是难民。

重点是：若一宗声请获得认可，难民的身份并不**取决于**最终认可ta为难民的那个决定。

但是，直到有关当局/联合国难民署最终做出决定之前，我们都不知道ta到底是不是难民

本网站中除非特别注明，否则所有的“难民申请”“庇护申请”实际上指的是“声请”。不便之处敬请原谅
